#### Dragon's Realm

<p>
    Dragon's Realm é um jogo do tipo RPG que tem como objetivo misturar batalhas com chat virtual entre usuários. A ideia central é desenvolver um game no qual eles possam realizar interações internas, trocas de itens, coleta de itens e assim sucessivamente. O game foi inspirado em diversos RPGs ao redor do mundo, tendo como principal referência o Tibia.
</p>
